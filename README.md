# CSCI 570 Fall 2025 Final Project: Sequence Alignment

## Project Overview
Implementation of Sequence Alignment using:
1. **Basic Algorithm:** Standard Dynamic Programming.
2. **Efficient Algorithm:** Memory-efficient DP with Divide and Conquer 

## Requirements
* **Python 3.x** (Tested on Python 3.12)
* No external libraries required (uses standard `sys`, `time`, `resource`).

## Instructions How to Run

### 1. Efficient Algorithm
We have created a wrapper script `efficient.sh` to compile/run the code as per project specs.

**Usage:**
```bash
./efficient.sh <input_file> <output_file>


./efficient.sh SampleTestCases/input1.txt output.txt

```

### 2. Basic Algorithm

**Usage:**
```bash
./basic.sh <input_file> <output_file>


./basic.sh SampleTestCases/input1.txt output.txt

```
